# Installation Guide.

### Fresh Install Info
	sudo pacman -S --noconfirm git stow
	mkdir /home/dotfiles/
	git clone 'https://github.com/niranjanshr13/dotfiles' /home/dotfiles/
	cd /home/dotfiles/
	stow -vt /home/$USER/ .
